## About This Portfolio

This HomeLab is the second iteration of my infrastructure lab and reflects the lessons learned from my original infrastructure-focused environment. 

The focus has shifted towards Windows administration, security hardening, automation and hybrid infrastructure concepts.

The goal is to build, operate, and document a practical enterprise-inspired environment while continuously expanding technical knowledge and administrative skills.
